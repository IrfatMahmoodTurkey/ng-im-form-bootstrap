# NG IM Form powered by Bootstrap

A lightweight Angular dynamic form builder powered by Bootstrap.

## Installation

Install the package along with Bootstrap and Font Awesome using npm, then include the required styles in your Angular project.

## Usage

Import the library modules into your Angular application and use the form builder/ form preview component in your templates.
